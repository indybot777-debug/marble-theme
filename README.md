# Marble Theme Refresh

Custom Shopify theme for marblesoda.jp

## Design Direction
- Forms.co clean catalog aesthetic
- Cav Empt subtle experimental touches  
- Monotone palette, muted/hollow colors
- Minimal navigation
- Grid-based product display

## References
- https://www.forms.co/ (grid, product codes, catalog vibe)
- https://www.cavempt.com/ (experimental visuals, floating products)
- Brain Dead, Stussy (cultural authenticity)

## Color Palette
- Background: #f5f5f5 (off-white, slight warmth)
- Text: #1a1a1a (near-black)
- Muted accent: #8a8a8a (gray)
- Subtle highlight: #e0e0e0

## Typography
- Clean sans-serif
- Tight letter-spacing for headers
- Generous line-height for readability

## Navigation
- Minimal top nav: Shop | Archive | World | Info
- Hidden cart indicator (appears on hover/add)
